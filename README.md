
# AutoML Streamlit App

A dataset-agnostic machine learning pipeline that automatically detects classification or regression tasks, preprocesses data, trains multiple models, and selects the best one.

## Features
- Detects classification or regression
- Handles missing values
- Handles class imbalance
- Trains multiple models
- Selects best model automatically
- Built with Streamlit UI

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
